My vim tips and tricks

How to run bash command over current file and replace buffer with result?

:!{cmd} sends {cmd} to the shell which executes it and shows its output on the screen.

:{range}!{filter} sends the lines from the current buffer inside {range} as the input of the {filter} program and replaces them with its output.

range

filter

:! By itself, runs the last external command (from your shell history)

:!! Repeats the last command

:silent !{cmd} Eliminates the need to hit enter after the command is done

:r !{cmd} Puts the output of $cmd into the current buffer.

return to vim after ctrl-z: fg
